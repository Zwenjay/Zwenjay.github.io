130 Surrounded Regions

Posted on 2017-10-24

Quesion

Given a 2D board containing ‘X’ and ‘O’ (the letter O), capture all regions surrounded by ‘X’.

A region is captured by flipping all ‘O’s into ‘X’s in that surrounded region.

For example,

X X X X
X O O X
X X O X
X O X X

After running your function, the board should be:

X X X X
X X X X
X X X X
X O X X

Solution 1 Union Find

First Union all ‘O’, then traverse the root array, once the point is at edge, make its root’s value special, then traverse it again, if ‘O’’s root is not special, turn it to ‘X’

Code

T & S Analysis

Time: Depends on Find time, suppose it’s f(n), then time is O( f(n) col row )

Space: O(row * col)

Solution 2 BFS

First, put all edge ‘O’ into queue, than do BFS, which could visit all ‘O’ that are related to edge ‘O’.
After that, traverse the visit array, if ‘O’ is not visited, turn it to ‘X’

Code

T & S Analysis

Time: O(row * col)

Space: O(row * col)
