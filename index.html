Question

A sequence of numbers is called a wiggle sequence if the differences between successive numbers strictly alternate between positive and negative. The first difference (if one exists) may be either positive or negative. A sequence with fewer than two elements is trivially a wiggle sequence.

For example, [1,7,4,9,2,5] is a wiggle sequence because the differences (6,-3,5,-7,3) are alternately positive and negative. In contrast, [1,4,7,2,5] and [1,7,4,5,5] are not wiggle sequences, the first because its first two differences are positive and the second because its last difference is zero.

Given a sequence of integers, return the length of the longest subsequence that is a wiggle sequence. A subsequence is obtained by deleting some number of elements (eventually, also zero) from the original sequence, leaving the remaining elements in their original order.

Output: 6

Output: 7

Input: [1,2,3,4,5,6,7,8,9]

class Solution {
    public int minSubArrayLen(int s, int[] nums) {
        int j, minL = nums.length+1;
        int[] sumArray = new int[nums.length + 1];
        sumArray[0] = 0;
        for(int i = 1; i < sumArray.length; i++)
            sumArray[i] = sumArray[i-1] + nums[i-1];
        for(int i = 0; i < sumArray.length; i++){
            if(sumArray[i] < s)
                continue;
            j = findLast(i, sumArray, sumArray[i] - s);
            minL = Math.min(i-j, minL);
        }
        if(minL == nums.length+1)
            return 0;
        return minL;
    }

    private int findLast(int hi, int[] nums, int target){
        int mid, lo = 0;
        while(lo < hi){
            mid = lo + (hi - lo)/2;
            if(nums[mid] > target){
                if(nums[mid-1] <= target)
                    return mid-1;
                hi = mid;
            }else{
                if(nums[mid+1]>target)
                    return mid;
                lo = mid;
            }
        }
        return 0;
    } 
}

T&S Analysis

Time: O(NlogN)

Space: O(N)
